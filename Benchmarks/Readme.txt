.asm file is the source code of Mars.
.hex file is the output code, which can be loaded into the rom or ram in the logisim.
The memory file exported from Mars must be added with 'v2.0 raw' in the first line, as shown in the following example:

v2.0 raw
00008020
2011001c
8e130000
8e340000
0274402a
11000002
ae330000
ae140000
2231fffc
1611fff8
22100004
